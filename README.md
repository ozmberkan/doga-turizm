## Kurulum
Projeyi klonlayarak başlayın:
```bash 
https://github.com/ozmberkan/doga-turizm
```
Gerekli paketleri yüklemek için:
```bash
npm install
```

Projeyi çalıştırmak için:
```bash
npm run dev
```
